This page will contain all the professor data, and will allow professors to login or create new professor profiles. Each professor will enter their ID number, name, department, and office building, and tenure status to be added to be the database. Login will be based on unique ID number. Professors may add/drop/view courses after logging in. Professors can also delete their profile after logging in. WIP.


Text entry fields for the following types of data: Login, ID number, name. Searchable dropdown menus for the following types of data: department, office building, tenure status.
